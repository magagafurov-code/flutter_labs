# Weather Mini — Погодное приложение

Мобильное приложение для просмотра текущей погоды в выбранных городах.  
Пользователь может добавлять города в избранное и быстро переключаться между ними для просмотра погодных условий.

---

## Основные функции

- Просмотр текущей погоды для выбранного города
- Отображение температуры, влажности и описания погоды
- Добавление новых городов в список сохраненных
- Переключение между сохраненными городами
- Хранение списка городов локально

---

## Экраны приложения

### Главный экран

- Название выбранного города
- Текущая температура
- Влажность воздуха
- Описание погодных условий
- Кнопка перехода к списку городов

### Экран списка городов

- Список сохраненных городов
- Поле ввода для нового города
- Кнопка «Добавить»
- Возможность выбора города для просмотра погоды

---

## Навигация

- Главный экран → Экран списка городов (кнопка перехода)
- Экран списка городов → Главный экран (при выборе города)

---

## Сценарии использования

### Добавление нового города
1. Пользователь открывает экран списка городов
2. Вводит название города в поле ввода
3. Нажимает кнопку «Добавить»
4. Город сохраняется в список

### Просмотр погоды в сохраненном городе
1. Пользователь открывает экран списка городов
2. Выбирает нужный город из списка
3. Открывается главный экран с погодой для выбранного города

### Обновление данных о погоде
- При открытии главного экрана всегда загружаются актуальные данные для выбранного города

---

## Используемые API

- OpenWeatherMap (текущая погода)
- WeatherAPI (альтернативный источник)
- Tomorrow.io (резервный вариант)

Для работы приложения требуется API-ключ от одного из провайдеров.
